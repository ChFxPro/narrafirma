Question textarea Enter a question to ask people about their stories.
Type select What type of question is this? The question types are:
  • boolean: yes and no choices
  • label: not a question, just text
  • header: same as a label, only in bold
  • checkbox: one check box (enter label in options)
  • checkboxes: a series of checkboxes (enter label in options)
  • text: a one-line free text field
  • textarea: a multi-line free text field
  • select: a drop-down box (enter choices in options)
  • radiobuttons: a set of mutually-exclusive radio buttons (enter choices in options)
  • slider: a range from 0 to 100 (enter end labels in options)
Short name text Enter a short name we can use to refer to the question. (It must be unique within the project.)
Options textarea If your question requires choices, enter them here (one per line).
Help textarea If you want to provide popup help to people answering the question, enter it here.
templateList "storyQuestions" You can copy a question from this list.

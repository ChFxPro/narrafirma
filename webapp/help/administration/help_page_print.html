Print

On this page you can print a project report. This page also serves to remind you of all the pages from which you can print things about your project.

What to do here

Print a project report or follow a link to print something else about the project.

Frequently-asked questions
Can I open a project report in my word processor?

Yes. The HTML document uses CSS styles, and they are included in the file (instead of referencing an external CSS file), so your word processor might be able to pick them up and use them as document styles. Even if the styles don't come out right, you will still be able to edit and style the text manually.

Can I print a project report, edit it, and then import it back into NarraFirma?

No. Unlike an exported file, a printed report cannot be read by NarraFirma. It is for your use outside the program.
